# Glossary

**ACK** — Acknowledgement flag used in TCP communication.

**APIPA** — Automatic Private IP Addressing. Windows may assign an address in `169.254.0.0/16` when DHCP fails.

**ARP** — Address Resolution Protocol, used to map IPv4 addresses to MAC addresses.

**Capture filter** — A filter applied before or during capture to limit recorded packets.

**Client** — A system or application that requests a service.

**DHCP** — Dynamic Host Configuration Protocol, used to provide IP configuration.

**Display filter** — A filter applied after capture to control which packets are shown.

**DNS** — Domain Name System, used to translate names into IP addresses.

**DORA** — Discover, Offer, Request, Acknowledge; the basic DHCP exchange.

**FIN** — TCP flag used during an orderly connection close.

**ICMP** — Internet Control Message Protocol, commonly used for ping and network errors.

**NXDOMAIN** — DNS response indicating that a requested domain name does not exist.

**Packet** — A formatted unit of network data.

**PCAP/PCAPNG** — Common packet-capture file formats.

**RST** — TCP reset flag used to immediately terminate or refuse a connection.

**SNI** — Server Name Indication, a TLS extension that identifies the requested hostname.

**SYN** — TCP synchronization flag used to begin a connection.

**TCP stream** — A complete TCP conversation between endpoints.

**TLS** — Transport Layer Security, used to encrypt application traffic.

**TTL** — Time To Live, an IP field reduced by each router hop.
