# Screenshots

Store sanitized screenshots in this folder.

## Recommended Naming

```text
lab-01-icmp-echo-request.png
lab-01-icmp-echo-reply.png
lab-02-dns-query.png
lab-02-dns-response.png
scenario-01-tcp-syn-timeout.png
```

## Screenshot Standards

- Crop to the relevant Wireshark area
- Hide unrelated tabs and windows
- Remove usernames, email addresses, tokens, and private hostnames
- Use readable zoom
- Add a short caption in the related Markdown file
- Do not use a screenshot as proof unless the packet details support the written conclusion
