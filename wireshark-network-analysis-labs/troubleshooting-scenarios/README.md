# Troubleshooting Scenarios

These files simulate common support cases. Each scenario is a documentation exercise and should be updated with actual lab evidence before being presented as completed.

## Scenario Workflow

1. Read the user-reported symptom.
2. Identify the likely protocols involved.
3. Reproduce the issue in a controlled lab.
4. Capture the traffic.
5. Apply relevant display filters.
6. Record packet evidence.
7. Separate confirmed findings from possible causes.
8. Recommend a resolution or escalation.
